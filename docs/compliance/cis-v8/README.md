# CIS v8 — Detailed Safeguard Library

This directory is reserved for **per-safeguard** implementation
details, including:

- Control implementation narratives.
- Assessment procedures (how an auditor would test the safeguard).
- Evidence-record samples.
- Compensating-control notes.

The authoritative cross-walk and high-level mapping is in
[`../cis-controls-v8.md`](../cis-controls-v8.md). This directory is
the "long form" — one file per safeguard, or grouped by Implementation
Group.

## Index

- IG1 (56 safeguards) — see `../cis-controls-v8.md` for the master table.
- IG2 (74 safeguards) — same.
- IG3 (23 safeguards) — same.

## When to add a file here

When a single safeguard is complex enough to warrant a multi-page
narrative (e.g., a full incident response runbook for CIS 17.4, or a
detailed architecture diagram for CIS 12.2), add it here. Otherwise,
keep the safeguard in the master table.
