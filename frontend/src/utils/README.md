# Frontend utilities (legacy)

Utilities in the new AionUi live under
[`frontend/src/lib/utils/`](../../lib/utils/). This directory is preserved for
legacy imports during the migration window (Sprint 2).

- `format.ts` — date, number, severity formatting
- `cn.ts` — className composition helper
- `debounce.ts`, `throttle.ts`
- `sort.ts`, `group-by.ts`
- `pagination.ts`
